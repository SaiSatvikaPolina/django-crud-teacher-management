# Teacher Management System

A full-stack **Teacher Management System** built with **Django** that enables efficient management of teacher records. The application provides CRUD functionality, image uploads, and a responsive user interface for managing teacher information.

## Features

- Add new teacher records
- View all teachers
- Update teacher details
- Delete teacher records
- Upload and display teacher profile images
- Responsive user interface using Bootstrap
- SQLite database integration
- Django ORM for database operations

## Tech Stack

- Python
- Django
- HTML5
- CSS3
- Bootstrap 5
- SQLite
- Django Template Language (DTL)
